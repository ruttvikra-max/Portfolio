# Visual Inspiration — Direction Notes

Two reference sets shared (editorial layouts). Images were shared inline; this captures the direction so we can rebuild it.

## Reference A — "Constructivismo / PYMES" editorial magazine spread
- Russian-Constructivist / Swiss editorial energy.
- Palette: **cream/off-white paper + black + a single hot red**.
- Huge condensed display type, uppercase headlines, bold rules/lines, stars and geometric blocks.
- Dense multi-column grids; big numbers used as graphic elements; red blocks behind key labels.
- Takeaway: confident, print-magazine feel. Type does the heavy lifting. One accent color only.

## Reference B — "Dmitri Prokofich Razumikhin" resume/portfolio template
- Same cream + black + red system, more modern/clean.
- Sectioned like our case study: `.ABOUT ME`, `.EXPERIENCE`, `.WORKS`, `.EDUCATION`, `.SKILLS`, `.PORTFOLIO`, `.CONTACT`.
- Leading dot prefix on section titles (`.SECTION`) — strong recurring motif we can adopt.
- Bold black slab headlines, red highlight blocks, percentage/stat bars, barcode + meta strip in hero.
- Large editorial image blocks alongside tight body text.
- Takeaway: this is the closest structural model — section-driven, stat-forward, editorial but legible.

## Direction we'll adopt
- **System:** cream background (`#FEF9EC`), near-black text (`#29303D`), red accent (`#DB3A34`). Yellow/orange/blue reserved for data viz.
- **Type:** Hecate Display for big headlines + stats; 001 Sans Serif for body.
- **Motifs:** `.SECTION` dot-prefix titles, oversized stat numbers, red highlight blocks, thin rules, meta strip in hero.
- **Layout:** editorial grid, generous whitespace, image blocks with text written *around* them (per brief: Challenge → Decision → Result).
- **Restraint:** one accent color as the spine; avoid rainbow. Keep it premium, not busy.
