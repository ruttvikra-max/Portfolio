# Color Palette — Rules Engine Case Study

Treated as a swappable asset (like the last case study). All values feed `site/src/styles/tokens.css`. Change here → propagate to tokens.

## Primary (accent / energy — used for emphasis, highlights, key numbers)
| Token | Hex | Use |
|---|---|---|
| `--color-primary-yellow` | `#EFB717` | Hero accents, highlight blocks, key stats |
| `--color-primary-orange` | `#EC700A` | Secondary emphasis, hover, callouts |
| `--color-primary-red` | `#DB3A34` | Strong emphasis, section markers, the "editorial red" of the inspiration |

## Secondary (trust / system — links, structural UI accents)
| Token | Hex | Use |
|---|---|---|
| `--color-secondary-blue` | `#2A6AF1` | Interactive, links, data viz |
| `--color-secondary-blue-deep` | `#0A399E` | Headers on light, deep accents |

## Tertiary (neutrals — text, dividers, surfaces)
| Token | Hex | Use |
|---|---|---|
| `--color-neutral-light` | `#B0B9C8` | Muted text, borders, dividers |
| `--color-neutral-mid` | `#465167` | Body secondary text |
| `--color-neutral-dark` | `#29303D` | Primary text, dark sections |

## Background
| Token | Hex | Use |
|---|---|---|
| `--color-bg` | `#FEF9EC` | Page background (warm cream — matches inspiration) |

## Notes
- Inspiration decks pair **cream background + near-black + a single hot red** with bold black type. Our red is `#DB3A34`; cream is `#FEF9EC`; near-black is `#29303D`.
- Yellow/orange/blue are available for data viz and accent variety but the editorial spine is **cream + dark + red**.
